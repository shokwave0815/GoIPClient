Über das Programm
----------------------

Der goIP-Client aktualisiert automatisch die IP-Adresse bei dem Dienst goIP.de. Voraussetzung ist nur ein gültiger Account bei dem Dienst.

Bedienung
------------

Nach Eingabe der Zugangsdaten (Benutzername und Passwort) kann man mittels "jetzt aktualisieren" die IP beim Dienst aktualisieren.
Das Programm überprüft alle 5 Minuten, ob die IP aktualisiert werden muss.
Unter "Optionen / Zeiteinstellung" kann festgelegt werden zu welchem Zeitpunkt eine Aktualisierung notwendig ist (Zwanstrennung vom Provider).
Unter "Optionen / Autostart" kann der automatische Start des Programms eingerichtet werden.

Beim minimieren des Programms erscheint ein Symbol im Systemtray (in der Taskleiste neben der Uhrzeit) über den es wieder angezeigt oder geschlossen werden kann.
Ist das Programm geschlossen werden keine Aktualisierungen ausgeführt.
